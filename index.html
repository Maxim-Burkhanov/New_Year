Годы

Чтобы сделать настроение новогодним, по настоящему теплым и у ютным нужно выполнить несколько простых действий. Можно начать с самого простого: начать украшать дом. Развесить по дому новогодние гирлянды, мишуру. Разложить цедру, корицу, гвоздику для создание запаха Нового Года.

    Украшение елки

  • Для создания новогодней атмосферы достаточно даже просто поставить небольшую елочку в квартире. Красиво украсить её: поставить звезду на макушку, повесить "дождик", шары, "сосульки". Подложить под неё разные подарки для гостей

    • Новогодние игры

  • Чтобы провести Новый Год не скучая нужно пригласить побольше друзей, родственников и подготовить игры. Отлично подойдут "Фанты". В этой игре нужно выполнять разные смешные задания. Еще можно поучаствовать в игре "Тайный Санта". В этой игре каждый делает подарок своему своему тайному другу. Каждый ведь хочет получить оригинальные и полезные подарки)

    • Новогоднее меню

  • Подготовьте Новогоднее меню! Заранее закупитесь в магазине нужными продуктами, приготовьте блюда для гостей.
